# Setup

## dw-api
Assumes user has python 3.8+ installed
1. pip install flask
2. pip install flask_cors
3. python process_data.py
4. python main.py

Api starts on http://localhost:5000/autosuggest

## dw-web
1. npm install
2. npm run build
3. npm run start

Go to http://localhost:3000 and start typing into the searchbox
